## Zabbix 4 - Docker-Compose
Docker-Compose that raises a Zabbix environment, ready to use.


### How do you use it?

#### Step 1 - Run the command below
```bash
docker-compose up -d
```

#### Step 2 - Open the link below
http://localhost:8888/

#### Step 3 - Enter the default credentials
**user**: Admin
**pass**: zabbix
